# Oasis Hub

Vite + React + TypeScript website for Annoor Education Family.

## Development

```sh
npm install
npm run dev
```

The local dev server runs on `http://localhost:3001`.

## Build

```sh
npm run build
```

The production output is generated in `dist/`.

## Test

```sh
npm run test
npm run lint
```

## Deploy

Deploy the project as a static Vite app with:

- Build command: `npm run build`
- Output directory: `dist`
